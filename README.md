# Tela de Login com Autenticação Simulada

Construção de um projeto de uma **tela de login moderna** desenvolvida com **HTML, CSS e JavaScript puro**, com foco em boas práticas de frontend e experiência do usuário.

---

## 🚀 Funcionalidades

✅ Login com validação  
✅ Cadastro de usuário (simulado)  
✅ Armazenamento com LocalStorage  
✅ Sessão de login persistente  
✅ Dark Mode (tema claro/escuro)  
✅ Botão de alternar tema (☀️/🌙)  
✅ Mostrar/Ocultar senha 👁️  
✅ Responsividade (mobile-friendly)  
✅ Proteção de rota (dashboard)

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- LocalStorage (Web API)
